Платья бывают очень разными, и их можно классифицировать по разными признакам: по силуэту, крою, посадке и длине. В этой статье представлены виды платьев, сгрупированные по этим признакам. Спойлер: будет очень много каруселей (целых 39)!

Виды платьев по силуэту

Силуэтом называют внешние очертания платья или другого предмета одежды. В его основе лежит простая геометрическая фигура. Кроме того, платья различаются по прилеганию: они могут быть облегающими, полуприлегающими и свободными.

Итак, по силуэту можно выделить следующие типы платьев:

  • Первый вид платья, который мы рассмотрим, - это прямое платье, то есть платье прямоугольной формы. {% assign carousel=site.data.dresses_carousels.items[0] %} {% include carousel.html title=carousel.title id=0 gallery_images=carousel.images gallery_sources=carousel.sources %}
  • Платье А-силуэта - это платье, зауженное в грудной области и равномерно расширяющееся к низу. Оно напоминает по форме букву А, поэтому и получило такое название; {% assign carousel=site.data.dresses_carousels.items[1] %} {% include carousel.html title=carousel.title id=1 gallery_images=carousel.images gallery_sources=carousel.sources %}
  • Платье-карандаш, или платье-футляр - плотно прилегающее платье с сужением книзу. Кстати, это хороший выбор для наряда в стиле ретро! {% assign carousel=site.data.dresses_carousels.items[2] %} {% include carousel.html title=carousel.title id=2 gallery_images=carousel.images gallery_sources=carousel.sources %}
  • Платье-тюльпан имеет форму перевёрнутого бутона тюльпана - отсюда и название. Оно заужено в области талии, а к бёдрам расширяется. Подол собирается при помощи резинки или тесьмы. {% assign carousel=site.data.dresses_carousels.items[3] %} {% include carousel.html title=carousel.title id=3 gallery_images=carousel.images gallery_sources=carousel.sources gallery_captions=carousel.captions %}
  • Платье-бочонок расширено в области талии, за счёт этого создаётся овальный силуэт. {% assign carousel=site.data.dresses_carousels.items[4] %} {% include carousel.html title=carousel.title id=4 gallery_images=carousel.images gallery_sources=carousel.sources gallery_captions=carousel.captions %}

Виды платьев по крою

Крой - это структура частей платья, основанная на чертеже для его пошива, то есть выкройке. Существует множество видов кроя. Иногда им дают названия по форме части ткани, из которой сделана деталь, или по способу соединения деталей платья.

Итак, перечислим основные виды кроя платьев (я не стала брать из источника те виды кроя, которые не могу отличить от уже упомянутых видов платьев, классифицированных по силуэту).

  • Платье-баллон - платье с присборенным на ленту или тесьму низом. {% assign carousel=site.data.dresses_carousels.items[6] %} {% include carousel.html id=6 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Кроп-топ - это по сути не платье, а топ и юбка. Вернее, это можно назвать платьем, разделённым на две части. {% assign carousel=site.data.dresses_carousels.items[7] %} {% include carousel.html id=7 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье шемиз - летнее платье из лёгкого материала. Его длина - до колена, и оно расширяется книзу (А-силуэт). То же самое, что и платье-рубашка. {% assign carousel=site.data.dresses_carousels.items[8] %} {% include carousel.html id=8 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-туника - это длинное свободное платье трапециевидного силуэта. Иногда комплектуется поясом. {% assign carousel=site.data.dresses_carousels.items[9] %} {% include carousel.html id=9 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-костюм - это комплект жакет + платье или пиджак + юбка, сшитый из лёгких тканей. {% assign carousel=site.data.dresses_carousels.items[10] %} {% include carousel.html id=10 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-пиджак можно рассматривать как что-то среднее между платьем и жакетом. {% assign carousel=site.data.dresses_carousels.items[11] %} {% include carousel.html id=11 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-трансформер - это платье, состоящее из вазимозаменяемых деталей. Своё название получило за способность менять длину, форму и стиль. {% assign carousel=site.data.dresses_carousels.items[12] %} {% include carousel.html id=12 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-футболку можно рассматиривать как длинную футболку с короткими рукавами. Обычно такая модель неприталенная. {% assign carousel=site.data.dresses_carousels.items[13] %} {% include carousel.html id=13 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-халат имеет особую застёжку-молнию, которая иногда выполняется в виде ложного нахлёста или пуговиц. {% assign carousel=site.data.dresses_carousels.items[14] %} {% include carousel.html id=14 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Отличительная черта платья-лапши - фактура вертикальных рубчиков. Оно делается из эластичного материала, облегает тело и обычно имеет большую длину. {% assign carousel=site.data.dresses_carousels.items[15] %} {% include carousel.html id=15 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-жилет по форме напоминает удлинённый фасон жилета. {% assign carousel=site.data.dresses_carousels.items[16] %} {% include carousel.html id=16 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Вообще, брюки-платье, или кюлоты представляют из себя комбинезон, но у него широкие брюки, вследствие чего эта модель похожа на платье. {% assign carousel=site.data.dresses_carousels.items[17] %} {% include carousel.html id=17 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-свитер подходит для холодного времени года. Как и платье-футболка, оно редко бывает приталенным. {% assign carousel=site.data.dresses_carousels.items[18] %} {% include carousel.html id=18 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-поло - это прямое платье длиной до колена, неотъемлемые атрибуты которого - воротник, рукава и небольшой вырез, застёгивающийся на несколько пуговиц. {% assign carousel=site.data.dresses_carousels.items[19] %} {% include carousel.html id=19 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-пальто может составить основу образа осенью. Его отличительные черты - длинные рукава, длина до середины голени или колена и нестандартный ворот. Разумеется, оно сшито из плотной ткани. {% assign carousel=site.data.dresses_carousels.items[20] %} {% include carousel.html id=20 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title gallery_captions=carousel.captions %}
  • Платье-водолазка - по сути, удлинённая модель тонкого облегающего свитера с высоким воротом. Может быть как из плотной, так из из достаточно лёгкой и эластичной ткани. {% assign carousel=site.data.dresses_carousels.items[21] %} {% include carousel.html id=21 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье фасона "летучая мышь" шьётся из простой трикотажной ткани. Отличается свободным кроем и постепенно сужающимися рукавами. {% assign carousel=site.data.dresses_carousels.items[22] %} {% include carousel.html id=22 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-толстовка и платье-худи весьма схожи, так что мы не будем рассматривать их отдельно. Как и у толстовки, у такого платья будет капюшон или объёмный воротник, оно будет тёплым, однако будет отличаться от толстовки длиной. {% assign carousel=site.data.dresses_carousels.items[23] %} {% include carousel.html id=23 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-комбинация - элегантное вечернее платье, похожее на разновидность женского нижнего белья, однако предназначенное для выхода в свет. {% assign carousel=site.data.dresses_carousels.items[24] %} {% include carousel.html id=24 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Отличительная черта платья-бандо - открытые плечи. В верхней части оно обязательно обтягивающее. {% assign carousel=site.data.dresses_carousels.items[25] %} {% include carousel.html id=25 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Верх у платья-облака облегающий, а юбка пышная. Подходит не для повседневной жизни, а для особого случая. Я думаю, его можно даже использовать как свадебное платье. {% assign carousel=site.data.dresses_carousels.items[26] %} {% include carousel.html id=26 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • За счёт того, что многоярусное платье состоит из нескольких рядов или деталей, создаётся интересный силуэт. {% assign carousel=site.data.dresses_carousels.items[27] %} {% include carousel.html id=27 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Как понятно из названия, цельнокроеное платье сождано из одной цельной части материала. При этом его силуэт может быть различным. {% assign carousel=site.data.dresses_carousels.items[28] %} {% include carousel.html id=28 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}

Виды платьев по крою верхней части

  • Одна часть платья с запахом находит на другую и держится таким образом за счёт пояса, вшитого с обеих сторон платья. {% assign carousel=site.data.dresses_carousels.items[29] %} {% include carousel.html id=29 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • У платья с напуском на талии верхняя часть свободно нависает над поясом. {% assign carousel=site.data.dresses_carousels.items[30] %} {% include carousel.html id=30 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}

Виды платьев по крою юбки

  • Нижняя часть платья с юбкой-солнце - это объёмная юбка с одним или двумя боковыми швами. Её конструкция на плоскости представляет собой круг, что и послужило причиной для названия. {% assign carousel=site.data.dresses_carousels.items[31] %} {% include carousel.html id=31 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Особенность юбки плиссе - мелкие, механически сжатые, непрошитые складки. Они могут быть плоскими или выступать гармошкой. Такая юбка может быть и низом платья. {% assign carousel=site.data.dresses_carousels.items[32] %} {% include carousel.html id=32 gallery_images=carousel.images gallery_sources=carousel.sources gallery_captions=carousel.captions title=carousel.title %}
  • С изнаночной стороны линия платья со шлейфом удлинена. {% assign carousel=site.data.dresses_carousels.items[33] %} {% include carousel.html id=33 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье годе (рыбка, русалка) - длинное, цельнокроеное платье с расклешённой юбкой ниже колена. Такой фасон часто выбирают невесты. {% assign carousel=site.data.dresses_carousels.items[34] %} {% include carousel.html id=34 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Платье-каскад отличается пышной, ассиметричной юбкой. {% assign carousel=site.data.dresses_carousels.items[35] %} {% include carousel.html id=35 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
  • Юбка платья-пачки сшита из нескольких слоёв лёгкой ткани. Обычно она состоит из 6-15 подъюбников, за счёт чего обладает пышностью. Прекрасный выбор для образа в стиле балеткор! {% assign carousel=site.data.dresses_carousels.items[36] %} {% include carousel.html id=36 gallery_images=carousel.images gallery_sources=carousel.sources title=carousel.title %}
